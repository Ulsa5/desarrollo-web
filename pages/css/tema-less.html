LESS - CSS CON UN POCO MAS


El concepto de la web ha evolucionado desde su concepción y ahora se refiere a una poderosa plataforma que soporta aplicaciones potentes para las organizaciones. Con una amplia gama de tecnologías disponibles, los profesionales deben tener conceptos sólidos que les permitan no solamente elegir las mejores tecnologías que necesiten sino poder diseñar sistemas que aprovechen los mejores conceptos y las mejores tendencias para ofrecer soluciones de vanguardia.

Al hablar de desarrollo web se debe enfocar en 2 áreas de dominio: desarrollo frontend y desarrollo backend, las cuales agrupan tanto conceptos como tecnologías para desarrollo de software que evolucionan constantemente y que tienden a especializar a los profesionales en una de ellas.
